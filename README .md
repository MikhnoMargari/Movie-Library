# Movie Library

**Автор:** Михно Маргарита
## Описание

Это простое GUI-приложение для ведения личной кинотеки. Позволяет добавлять фильмы, фильтровать их по жанру и году, сохранять и загружать данные в JSON-файл.

## Использование

1. Запустите программу.
2. Введите название, жанр, год выпуска и рейтинг фильма.
3. Нажмите «Добавить фильм» для добавления.
4. Используйте фильтры для поиска по жанру и году.
5. Все данные сохраняются автоматически при добавлении.

## Возможные тесты

- Добавление фильма с некоррректным годом или рейтингом — появится сообщение об ошибке.
- Фильтрация по жанру и году.
